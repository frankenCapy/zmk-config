05

the change

▸ drop the positional gate · keep the gatekeeper
behaviors {
    hml: home_row_mod_left {
        compatible = "zmk,behavior-hold-tap";
        flavor = "balanced";            // keep · the tiebreaker
        require-prior-idle-ms = <150>;   // keep · the real protection ★
        tapping-term-ms = <280>;        // keep
        quick-tap-ms = <175>;           // keep · tap-then-hold repeat
        bindings = <&kp>, <&kp>;
        hold-trigger-key-positions = <6 7 8 9 ...>;   // ✗ remove · blocks same-hand chords
        hold-trigger-on-release;                       // ✗ remove
    };
    // …mirror the same removal on hmr (right hand)
};

Result: GUI+F (and every same-hand mod) fires the moment you pause and reach for it — while require-prior-idle keeps your fast typing free of misfires. Fully reversible via git if the feel is off.
